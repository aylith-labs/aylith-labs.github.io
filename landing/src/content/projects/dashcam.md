## Vision

Dashcam is a flight recorder for AI coding agents. It captures every event, screenshot, video, and decision from autonomous coding sessions, then presents them in a scrubable timeline you can review later — like YouTube for your agents' work. Come back to an overnight run and watch what happened instead of reconstructing it from diffs.

## The Problem

Developers increasingly run AI coding agents for hours at a time, often unattended. When they return, they have no way to see what happened beyond a git diff — no reasoning at each decision point, no record of what the app looked like at each stage, no view of test runs as they happened, and no timeline across parallel agents. The work happened; the visibility didn't.

## Key Differentiators

- **A timeline, not a diff**: scrub through the whole session, milestone by milestone.
- **Sees what the agent saw**: screenshots and video captured automatically as work progressed.
- **Built for many agents**: swim lanes show parallel runs side by side.
- **Replay any moment**: dockable panels reconstruct code, chat, and app state at any point in time.
